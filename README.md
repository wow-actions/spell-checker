## ðŸš€ Usage

Create a `.github/workflows/spell-checker.yml` file in the repository you want to install this action:

```yml
name: Spell Checker
on: push
jobs:
  run:
    runs-on: ubuntu-latest
    steps:
      - uses: wow-actions/spell-checker@v1
        with:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          include: |
            **/*.md
            **/*.txt
```
